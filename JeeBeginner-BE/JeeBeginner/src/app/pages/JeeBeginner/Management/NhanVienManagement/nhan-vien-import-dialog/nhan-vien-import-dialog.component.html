Import nhân viên từ Excel

Hướng dẫn:

1. Tải file mẫu, điền dữ liệu từ dòng thứ 2 và không đổi tên các cột.

2. Cột bắt buộc: Mã NV, Họ tên. File chỉ nhận định dạng .xlsx.

3. Mỗi dòng hợp lệ sẽ được tạo mới và mã hóa Họ tên, CCCD. Dòng trùng Mã NV sẽ được báo lỗi.
